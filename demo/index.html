Click each method to change it, click the map to re-generate. This demo requires the Unity Web Player. 

var generator = new DungeonGenerator();
generator.GenerateA()
         .()
         .ABitRandom()
         .SomewhatSparse()
         .WithMediumChanceToRemoveDeadEnds()
         .WithMediumSizeRooms()
         .WithLargeNumberOfRooms()
         .Now();
